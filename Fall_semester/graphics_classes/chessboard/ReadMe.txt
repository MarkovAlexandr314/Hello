Рисуем шахматную доску:

1) Добавить кнопку Quit.
2) Написать клетку-кнопку Cell, завести объект, отрисовать.
3) Составить доску из клеток, нарисовать, пощёлкать.
4) Добавить подписи на доску, цифры, буквы. Нарисовать.
5) Добавить реакцию на клик мышкой: активировать (выделить) по первому щелчку,
деактивировать (снять выделение) по второму щелчку, переместить выделение, если
кликнули в другую клетку. Белые клетки запретить выделять (логика шашек)
